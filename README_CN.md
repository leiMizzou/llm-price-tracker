# 💰 llm-price-tracker

LLM定价对比工具。跟踪和比较50+模型的价格。

## 功能特点

- **50+模型数据库** — 覆盖所有主流提供商
- **成本计算器** — 按Token用量估算费用
- **价格提醒** — 模型降价通知
- **历史追踪** — 价格变动记录

## 快速开始

```bash
python llm_prices.py compare gpt-4 claude-3-opus
python llm_prices.py calculate --tokens 1000000 --model gpt-4
python llm_prices.py list --sort price
```
